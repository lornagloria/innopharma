# innopharma
 Web application that allows registered pharmacies to self-register their medecine availablity and enables patients to search and locate pharmacies that stock their medecine. The app will enable its users to locate the products present in the pharmacy in  real time. This project is sstill currently in development and the structure may change as progress continues.


Technologies:

We will use the following softwares installed locally:
 * Node.js (LTS Version 18+): Includes the Node Package Manager (npm) for installing dependencies.
 * MongoDB: The database server must be running either locally or via a cloud service (e.g., MongoDB Atlas).
 * Git: For cloning the repository.
